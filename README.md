TrendSphere Shopping

TrendSphere Shopping is a modern, premium e-commerce web application that allows users to browse, shop, and explore a wide variety of products in categories like Fashion, Electronics, Beauty, Groceries, and more. Built with React, Material-UI, and modern libraries, it provides a clean, professional, and responsive shopping experience.

Features
Responsive Navigation
Top navbar with links to main sections
Shop by Categories drawer with professional icons
Category Browsing
Browse products by categories: Fashion, Electronics, Accessories, Bags, Footwear, Groceries, Beauty, Wellness, Jewellery
Clickable categories open dedicated pages
User Account Actions
Login and Wishlist features
Shopping Cart with badge count
Search Functionality
Real-time search for products
Professional UI/UX
TailwindCSS for styling
Material-UI components for icons and drawer
Hover effects for buttons and links
Free Shipping Info
Highlighted banner for promotional info
Technologies Used
Frontend: React, React Router DOM
Styling: TailwindCSS, Material-UI
Icons: React Icons, Material-UI Icons
State Management: React useState
Routing: React Router DOM
